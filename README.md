# Clad

**AI Personal Stylist & Wardrobe OS**

Clad is your personal AI stylist that knows every item in your closet, learns your personal taste, and delivers daily outfit recommendations from what you already own — then tells you exactly what to buy next to fill wardrobe gaps.

## How It Works

1.  **Scan**: Use vision AI to catalog your entire wardrobe from photos.
2.  **Learn**: Our AI learns your personal style, preferences, and lifestyle.
3.  **Style**: Receive daily outfit recommendations tailored to your taste, weather, and schedule.
4.  **Shop**: Identify wardrobe gaps and get curated purchase recommendations.

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: FastAPI (Python), PostgreSQL/SQLite
- **AI**: GPT-4 Vision, Custom Recommendation Engine

## Quick Start

### Backend
1. Navigate to `backend/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the server: `uvicorn app.main:app --reload`

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
